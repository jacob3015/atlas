# Atlas

A Python CLI toolkit for personal investment research.

Atlas is a command-line toolkit that automates market data collection, stock and ETF analysis, macro monitoring, and investment workflow.

The project is designed to support data-driven investment research by combining publicly available market data with reproducible analysis.

> **Status:** 🚧 Early development. Features and documentation will continue to evolve.

## Data Sources

Atlas collects publicly available market data from third-party providers.

Whenever possible, official APIs are preferred. Some providers are accessed through publicly available web endpoints.

Users are responsible for complying with the terms of service and licensing of each data provider.
